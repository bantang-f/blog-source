实际问题描述

昨天，我们文科班的宣传委员要做一期关于《全唐诗》的黑板报，其中有一部分需要按诗人的作品数量来列出排名。

作为文科班的编程老师，今天我来带你用Python做统计。

零基础编程

首先通俗地解释下什么是程序：

一组用某种语言编写的计算机可以运行的指令。

话不多说，开始零基础编程。

我们发现卷名、诗名和作者处于同一行并且格式特殊：

打开全唐诗文本文件，

file = open('Tang-Poetry.txt')

line = file.readline()

如果符合特殊格式，则我们可以提取作者信息。使用书名号】对这行文字进行划分，符号之后就是作者。

最后我们进行排序并打印出结果，

上面的代码和描述的解题思维有一些出入，那是因为需要对一些细节进行处理。

  • strs = line.strip().split("】")每行文本末有看不见的换行符号，才使得文本可以换行显示。 strip()方法可以去掉每一行末的换行符号。

  • if len(strs) == 2:用书名号划分后，还需要判断此行文本划分后的集合个数是否为2。这是因为有些作品没有名称和作者信息，如109604行划分后就不会形成2部分，而不需要处理此行。

  • sorted(statistics.items(), key=operator.itemgetter(1), reverse=True)排序的依据是作者的作品数，也就是statistics中元素的第二个值，并且从高到低的顺序。统计元素的第一个值为作者名称。python语言中计数标记从0开始，排序默认是降序。

计算机思维

对于实际问题，我们需要从计算机的角度去分析。把解决思路指令化，并转化编程语言。

一个常规的程序流程分为:

  • 输入数据
  • 处理数据
  • 输出数据

在今天的统计问题中，我们的数据源是一个txt文本。程序读取每一行，进行判断并更新统计数据。最后显示排序后的统计。

Python语法知识点

更加详细的语法知识可以到官方文档网站去查询，此章只列出在程序中出现的用法。

  • open方法用来读或者写文件
  • statistics = {}申明并初始化了一个字典数据。它可以像字典一样储存数据，如statistics["Li Bai"] = 1
  • while condition 是一个循环语句。在condition为真时，反复执行循环内的代码块。break指令也可以退出循环。
  • if condition 是一个判断语句。在conditon为真时，执行代码块，否则执行else的代码块。
  • len方法可以统计数据的长度。
  • sorted方法用来排序，并可以指定排序的依据和顺序。
  • print方法用来打印数据
